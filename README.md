# Demo Nursing Handoff

Pure frontend demo for an AI-assisted nursing handoff risk console.

## Local Preview

Open `index.html` in a browser, or serve the folder with any static server.

```bash
python3 -m http.server 4177
```

## GitHub Pages

This repository includes a GitHub Actions workflow at `.github/workflows/pages.yml`.
After pushing to `main`, GitHub Pages can deploy the static site from the workflow artifact.
